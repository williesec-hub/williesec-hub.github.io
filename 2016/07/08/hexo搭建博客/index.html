node.js+hexo+git+github搭建博客

  • 首先下载安装node.js+git

git

  1. ssh-keygen -t rsa -C “williesec@yeah.net”
  2. 将生成的id_rsa.pub中的内容拷贝到github中

  3. ssh -T git@github.com

    Hi cnfeat! You’ve successfully authenticated, but GitHub does not provide shell access.

hexo

  • 然后安装npm install -g hexo

    hexo init //生成相关文件
    hexo generate //创建静态网页文件
    hexo server //启动本地hexo服务，通过4000端口访问

github

  • 创建库名称为：用户名.github.io
  • 在库的setting中添加 域名不需要www

域名商网站中添加域名指向

  • CNAME @ username.github.io
  • CNAME www username.github.io
